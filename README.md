# github-bio

A simple web application that allows users to view GitHub profiles by entering a username. The application fetches user data from the GitHub API and displays the user's avatar, name, bio, followers, and following count.

## Features

-  Fetch and display GitHub user profiles.
-  Show user avatar, name, and bio.
-  Format followers and following counts for better readability.

## Technologies Used

-  HTML
-  CSS
-  JavaScript
-  GitHub API

## Usage

1. Enter a GitHub username in the input field.
2. Click the "Search" button to fetch the profile.
3. The user's information will be displayed below the input field.
